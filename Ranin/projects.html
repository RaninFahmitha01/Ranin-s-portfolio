Weather App

This weather application allows users to search for real-time weather information by city name. Built using HTML, CSS, and JavaScript, it fetches data from a public weather API (like OpenWeatherMap) and displays temperature, humidity, and weather conditions dynamically. It includes responsive design and error handling for invalid inputs.

Tic-tac-toe

A classic two-player Tic-Tac-Toe game created using HTML, CSS, and JavaScript. The game features a simple and responsive interface, interactive game board, turn-based logic, win/draw detection, and a restart button. Great for demonstrating JavaScript DOM manipulation and game logic implementation.

stopwatch

This stopwatch application is developed with HTML, CSS, and JavaScript. It includes start, stop, and reset functionalities along with a real-time counter that tracks hours, minutes, and seconds. The UI is clean and minimal, with dynamic updates using JavaScript timers.
